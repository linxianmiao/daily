卡片信息结构

Explore 页面 · 每张词条卡片的数据展示结构

必填 始终展示
可选 有值才展示
切换 中英互换
计算 派生值
未展示 数据库有但未用
Swiper 卡片
keyword 必填 词条名称，大字标题展示（96rpx）
pinyin 可选 拼音注音，有值时显示在标题下方
遍历 definitions[] 数组 — 每个释义生成一张卡片
Definition 卡片
defIdx 计算 序号图标 + "Definition N" 标签，颜色按奇偶交替
MEANING 区域
definition_us 切换 英文释义，默认展示。showZh=false 时显示
definition_zh 切换 中文释义，点击翻译按钮后显示。showZh=true 时显示
⚡ 回退逻辑：若对应字段为空，回退到 definition 字段
EXAMPLE 区域
📋 展示条件：example_us || example_zh || example 任一有值
example_us 切换 英文例句。showZh=false 时显示
example_zh 切换 中文例句。showZh=true 时显示
example 可选 通用例句，作为 example_us / example_zh 的回退
卡片底栏
showZh 计算 翻译按钮 [EN/中]，控制中英切换状态
🔘 展示条件：definition_zh && definition_us 都存在时才显示
thumbsUp 可选 点赞数 👍，格式化为 k 单位（如 1.2k）
🔢 派生字段：thumbsUpText = thumbsUp >= 1000 ? (n/1000).toFixed(1)+'k' : n
↑ 上滑查看下一个词条
📦 数据库中存在但未展示的字段
_id 未展示 数据库唯一标识，仅用于去重（seenIds）
category 未展示 词条分类，默认 'General'，未在 UI 中使用
source 未展示 数据来源，未在 UI 中使用
